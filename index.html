Estruturas de Dados

Pilha

A Pilha é uma estrutura de dados caracterizada por implementar o que chamamos de FILO: First In Last Out. Ou seja, o primeiro item adicionado à pilha deve ser o último a deixá-la. Aqui fizemos uma implementação utilizando um Objeto JavaScript.

Fila

A Fila é uma estrutura de dados com característica inversa à Pilha e implementa o que chamamos de FIFO: First In First Out. Ou seja, o primeiro item adicionado à fila deve ser o primeiro a deixá-la. Aqui fizemos uma implementação utilizando um Objeto JavaScript.

Algoritmos

Conversor de Base Decimal-N

Converte um número da base decimal para um de base-N entre 2 e 36 (inclusos). O procedimento é realizado dividindo o número inicial sucessivamente até encontrarmos o número 0. A cada divisão guardamos o resto da divisão (módulo) numa Estrutura de Pilha. Por fim, utilizamos Stack.pop() sucessivamente para acrescentar o valor, convertido para 0-Z, a uma string e a retornamos.
